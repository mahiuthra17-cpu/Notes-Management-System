# Notes Management System

## Project Overview
This project is a simple Notes Management System developed using Java, HTML, and CSS. It allows users to create and manage notes through a simple interface and displays a successful message after adding a note.

## Technologies Used
- Java
- HTML
- CSS

## Features
- Notes Management Form
- Enter Note Title
- Enter Note Description
- Add New Note
- View Notes
- Update Notes
- Delete Notes
- Simple and Responsive User Interface
- Displays Note Added Successfully Message

## Note Details
The application collects the following information:
- Note Title
- Note Description

## Project Structure
- MainActivity / NoteServlet (Java)
- index.html
- style.css

## Output
### Notes Management Form
- User enters note details.
- Clicks the **Add Note** button.
- The application displays a **Note Added Successfully** message.
- The entered note information is shown on the screen.
